# expo-redbook

## 功能清单

- [x] 自定义tabs
- [x] 二次封装fetch请求
- [x] 自定义图片预览组件
- [x] 支持瀑布流组件
- [x] 自定义下拉弹窗
- [x] 自定义抽屉组件
- [ ] 自定义原生Loading组件和Toast组件
- [ ] 支持操作SqLite数据库
- [ ] 支持expo-router的权限组件

## 安装

```bash
  npm install & cd ./mock-server & npm install
```

## 运行

```bash
 npm start & npm dev
```
